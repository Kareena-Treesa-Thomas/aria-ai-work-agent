# ARIA Demo Script

## Opening

ARIA is an Automated Response Intelligence Agent. It helps users dump raw work input into one place and get a useful next step.

## Demo 1: Email

Paste:

```text
Hi, can we move tomorrow's meeting to 3 PM? Also please send the revised proposal before then.
```

Click `Email`, then `Process with ARIA`.

Show:

- `REPLY`
- `TONE`
- `URGENCY`

## Demo 2: Meeting Notes

Paste:

```text
Product sync: Priya owns the landing page copy by Monday. Arun will fix the webhook. Decision: launch the demo build tonight. Follow up with design after testing.
```

Click `Meeting Notes`, then `Process with ARIA`.

Show:

- `DECISIONS`
- `TASKS`
- `DEADLINES`
- `FOLLOW-UP`

## Demo 3: Voice Memo

Paste:

```text
Quick note. We need to tell the team the customer wants a shorter onboarding flow. Ask engineering for feasibility and send a summary to the founders.
```

Click `Voice Memo`, then `Process with ARIA`.

Show:

- `SUMMARY`
- `ACTION ITEMS`
- `SEND TO`

## Closing

ARIA turns messy inputs into structured next actions. It is built as a functional prototype with a real backend path for OpenAI, n8n, Notion, and Twilio.
